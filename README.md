This project is part of the Coffee Shop Website initiative, designed to showcase the shop's offerings, enhance customer engagement, and enable online ordering 
## Project Description
The website consists of several key pages to deliver a seamless and engaging user experience for visitors and registered users.

### Main Pages
1. **Home Page (Landing Page)**
   - Hero section with a cozy tagline: *"Brewed with Passion, Served with Love."*
   - Menu Highlights showcasing bestsellers.
   - Call-to-Actions: *View Menu, Order Now, Find Us.*
   - Special Offers and Customer Reviews carousel.

2. **Menu Page**
   - Interactive, categorized menu with images, descriptions, and prices.
   - Customization options (e.g., milk types, add-ons).
   - Allergen Information and promotional labels like "New" or "Popular."

3. **Online Order Page**
   - Step-by-step ordering with customization and pickup/delivery options.
   - Real-time preparation timer.
   - Loyalty Program integration and diverse payment options.

4. **About Us Page**
   - Story section about the coffee shop's journey.
   - Core values like sustainability and community.
   - Photos and bios of the baristas.

5. **Contact Us Page**
   - Feedback form for suggestions.
   - Live chat support for order queries.
   - Links to social media platforms.
